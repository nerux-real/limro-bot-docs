Welcome to LimroBot documentation!
===================================

**LimroBot** is a multifunctional Discord bot made by py-cord
Thank you for visiting this site and going to use LimroBot to the fullest extent.
These docs will tell you how to use and configure the bot so that you can use all of its features without restriction.
Now the project is in beta stage, there may be some bugs, about which you can send me information to e-mail: limrostudios@gmail.com or write about it using /report command.
Our webpage: https://bot.limro.top

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project and docs are under active development.
